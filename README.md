# Calculadora-de-retencoes

## Um programa que calcula retenções federais sobre serviços tomados (IRRF e CSRF) e salva os cálculos para consulta
- O usuário deve incluir o nome do prestador de serviço, o serviço que está sendo prestado e o valor

- O programa irá verificar se o valor do serviço gera retenção a recolher, irá realizar um cálculo (O valor do serviço X as alíquotas de cada imposto, CSRF 4,65% e IRRF 1,5%)

- Se o valor das retenções ficar acima de 10,00 aparecerá na consulta o valor de cada imposto, se ficar abaixo, o sistema irá responder que não é devido

- Programa desenvolvido utilizando HTML e Javascript
